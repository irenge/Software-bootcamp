2. What is a DBMS and what are its advantages?
  DBMS stands for Database Management Systems. It is a collection of programs, which works to manage the database structure and control access to the data stored in the database.
The advantages of a DBMS are :
* Better data sharing: DBMS allows end-users to have more efficient access to data, it does so by managing the database and colling access to data.

* Improved data integration: DBMS helps provide a clearer and more integrated view of the organisationâ€™s operations to the end-users
* Minimised data inconsistency: DBMS greatly reduces the probability of data inconsistency
* Improved data access: The DBMS makes it possible to produce quick answers to query
* Improved decision making: Decisions are made based on information which DBMS generates at very high quality
* Increased end-user productivity: with DBMS there is a constant availability of data and ability to transform data into usable information. This fact improved end-user productivity.
